grepコマンドの使い方を覚えるために問題を解いてみましょう。
postkobe.csvは神戸市の郵便番号の一覧です。

郵便番号 = "Post code"
都道府県名 = "Prefecture name"
市名 = "City name"
区名 = "Ward name"
町名 = "Town name"

grepコマンドを用いて、以下の検索を行ってください。

1.住所にアルファベットの T を含むものを検索してください。
2.住所にアルファベットの W もしくは w を含むものを検索してください。
3.郵便番号が数字の 652 で始まるものを検索してください。
4.町名 がアルファベットの T で始まるものを検索してください。
5.市名 と 町名 が共にアルファベットの K で始まるものを検索してください。
6.町名 がちょうど 4 文字であるものを検索してください。
7.郵便番号が 0 で終わるものを検索してください。
8.末尾がピリオド(.)でおわるものを検索して下さい。
