grepコマンドの使い方を覚えるために問題を解いてみましょう。
postkobe.csvは神戸市の郵便番号の一覧です。

郵便番号 = "Post code"
都道府県名 = "Prefecture name"
市名 = "City name"
区名 = "Ward name"
町名 = "Town name"

grepコマンドを用いて、以下の検索を行ってください。

1.住所にアルファベットの T を含むものを検索してください。
2.住所にアルファベットの W もしくは w を含むものを検索してください。
3.郵便番号 が数字の 652 で始まるものを検索してください。
4.郵便番号 の末尾が 11 で終わるものを検索してください。
5.町名 がアルファベットの T で始まるものを検索してください。
6.市名 と 町名 が共にアルファベットの K で始まるものを検索してください。
7.町名 がちょうど 4 文字であるものを検索してください。
8.町名 が suji で終わるものを検索してください。
9.郵便番号 が数字の 652 で始まり 町名 が dai 終わる行を検索してください。
10.行の末尾が ピリオド(.) で終わるものを検索してください。
